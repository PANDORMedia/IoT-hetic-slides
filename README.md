# IoT — Arduino & ESP32

Slides de formation IoT couvrant les bases Arduino/ESP32, GPIO, capteurs et communication.

## Contenu

- **Jour 1** : Histoire IoT, syntaxe Arduino, simulation Tinkercad
- **Jour 2** : GPIO réel sur ESP32, anti-rebond, machine à états

## Utilisation

Ouvrir `slides/index.html` dans un navigateur.

### Navigation
- `←` `→` : slides précédente/suivante
- `N` : notes orateur
- `F` : plein écran
- `1` `2` : accès direct Jour 1/2
